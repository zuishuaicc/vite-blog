## vite 项目中使用 tailwind

### 1. 安装 tailwind 依赖

```bash
npm install -D tailwindcss postcss autoprefixer
```

### 2.初始化 postcss.config.js 文件

```bash
npx tailwindcss init -p
```

### 3.tailwind.config.js 中配置模版文件路径

```js
/** @type {import('tailwindcss').Config} */
export default {
  content: ["./index.html", "./src/**/*.{js,ts,jsx,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

### 4.在./src/index.css 文件中添加 @tailwind 指令.

```scss
@tailwind base;
@tailwind components;
@tailwind utilities;
```
